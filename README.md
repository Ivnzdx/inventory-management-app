# StockMaster Pro - Inventory Management App

Aplikasi manajemen inventaris barang berbasis web yang mempermudah pencatatan stok produk secara real-time.

## Fitur Utama
* **Manajemen Produk:** Tambah, edit, dan hapus data produk dengan mudah.
* **Status Stok Otomatis:** Menampilkan indikator status produk secara langsung (misal: STABLE).
* **Batas Stok Minimum:** Membantu memantau kapan barang harus segera di-restock.

## Teknologi Yang Digunakan
* React.js
* Vite
* Tailwind CSS / CSS Custom
